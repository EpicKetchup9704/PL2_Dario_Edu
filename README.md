# Proyecto final de Java - Creación de JavaFit
Este es el proyecto final de Java, hacer una aplicación OOP con una interfaz gráfica y serialización de los archivos, para que guarde los cambios. Hay que hacer un documento que resuma el trabajo y muestre sus características, hacer el Javadoc y hacer un manual para usuario.

 - Para los diagramas se recomienda usar una herramienta llamada UMLet, una app FOSS que permite hacer diagramas UML de forma más sencilla y más liviana.
 
 ## Descripción del proyecto
 JavaFit es una herramienta que permite administrar de forma sencilla un gimnasio virtual. Cuenta con un archivo serializado (donde se guardan los cambios); varias ventanas de Java Swing (login, registrarse, administrar) que permiten usar la aplicación de una forma amigable para el usuario.
